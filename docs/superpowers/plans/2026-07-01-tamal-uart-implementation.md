**Goal:** A compile-time-configurable 8N1 UART (RX + TX) for the host transport, single `Dom100` domain, 16× oversampling from an NCO enable — verified by a byte-exact TX→RX loopback.

**Architecture:** Four modules under `Tamal.Uart.*`: `BaudGen` (NCO `oversampleTick`), `Rx` (`uartRx`), `Tx` (`uartTx`), and the `uart` umbrella. RX/TX are Mealy machines gated by the shared oversample tick; the NCO is a phase accumulator. Tests drive the sequential logic with Clash `sampleN`/`fromList`.

**Tech Stack:** Clash 1.10 (`clash-prelude`), GHC, Stack; tasty + tasty-hunit + tasty-hedgehog + hedgehog; fourmolu (2-space, leading commas). Source of truth: `docs/superpowers/specs/2026-07-01-tamal-uart-design.md`.

**Build order rationale:** BaudGen → **Rx** → **Tx** → uart. `uartRx` is testable standalone (drive `tick = pure True` with a crafted line waveform); `uartTx` is then verified by feeding its line into the now-existing `uartRx` (fast loopback); `uart` closes the loop with the real NCO tick.

### Gotchas (read before starting)

- **`NumericUnderscores`** is not global; add `{-# LANGUAGE NumericUnderscores #-}` per UART file (as `Domain.hs` does) to write `2_000_000` / `100_000_000`.
- **`msb`/`lsb`** exist in `Clash.Prelude` (`Crc.hs` already uses `msb`). Use `lsb` for the TX data bit.
- **NCO feedback** uses the idiomatic `acc = register 0 acc'` self-reference — Clash breaks the loop through the register.
- **RX/TX lift**: `mealy step init (bundle (…)) |> unbundle` — `bundle`/`unbundle`/`mealy` from `Clash.Prelude`.
- **Tests are simulation, not pure functions**: `sampleN n sig` runs `sig` under a generated clock/reset/enable; `fromList xs` is an input `Signal` (use `xs ++ repeat …` so it never runs dry). `sampleN`'s cycle 0 is the reset cycle — pad inputs with idle and use tolerances/`repeat`.
- **`SNat @2_000_000`** passes the baud as a singleton; `snatToNum` reads it at compile time. `DomainPeriod dom` (from `KnownDomain dom`) gives the clock period in ps → `f_clk`.

- [ ] **Step 4 (HUMAN): implement `oversampleTick`**

Walkthrough (delivered live): the DDS/NCO idea — add `inc = 16*baud` to an accumulator each clock; when it reaches `f_clk`, subtract `f_clk` and emit a tick. That yields exactly 16 ticks per 50-clock bit. `f_clk` comes from `DomainPeriod dom`; the `acc = register 0 acc'` self-reference is the Clash feedback idiom; `mux tick a b` selects the wrapped vs unwrapped next value.

Reference (replace the stub body):

```haskell
oversampleTick baud = tick
 where
  periodPs :: Integer
  periodPs = snatToNum (SNat @(DomainPeriod dom))
  fClk :: Unsigned 32
  fClk = fromInteger (1_000_000_000_000 `div` periodPs)
  inc :: Unsigned 32
  inc = fromInteger (16 * snatToNum baud)

  acc :: Signal dom (Unsigned 32)
  acc = register 0 acc'
  sums = (+ inc) <$> acc
  tick = (>= fClk) <$> sums
  acc' = mux tick (subtract fClk <$> sums) sums
```

Walkthrough (live): the 2-flop synchronizer (`register high` twice) on the async line; the FSM steps only on ticks; `rxCnt :: Index 16` walks each bit; capture the synced line at counts 7/8/9; decide at count 15 with `maj`; assemble LSB-first by inserting each bit at the MSB and shifting right; `RxStop` emits `Just byte` on a high stop bit, else strobes framing error and drops the byte.

Reference (replace the stub + add the helpers `maj`, `rxStep`, `captureSample`, `decideBit`):

```haskell
uartRx tick rxLine = unbundle (mealy rxStep initRx (bundle (tick, synced)))
 where
  sync1 = register high rxLine
  synced = register high sync1
  initRx = RxS RxIdle 0 0 0 0 0

maj :: Bit -> Bit -> Bit -> Bit
maj a b c = (a .&. b) .|. (a .&. c) .|. (b .&. c)

rxStep :: RxS -> (Bool, Bit) -> (RxS, (Maybe (BitVector 8), Bool))
rxStep s (tick, line)
  | not tick = (s, (Nothing, False))
  | otherwise = case rxState s of
      RxIdle
        | line == low -> (s{rxState = RxStart, rxCnt = 0}, (Nothing, False))
        | otherwise -> (s, (Nothing, False))
      _ ->
        let s1 = captureSample s line
         in if rxCnt s == maxBound
              then decideBit s1
              else (s1{rxCnt = rxCnt s + 1}, (Nothing, False))

captureSample :: RxS -> Bit -> RxS
captureSample s line = case rxCnt s of
  7 -> s{rxS7 = line}
  8 -> s{rxS8 = line}
  9 -> s{rxS9 = line}
  _ -> s

decideBit :: RxS -> (RxS, (Maybe (BitVector 8), Bool))
decideBit s = case rxState s of
  RxStart
    | bit == low -> (s{rxState = RxData 0, rxCnt = 0}, (Nothing, False))
    | otherwise -> (s{rxState = RxIdle, rxCnt = 0}, (Nothing, False))
  RxData i ->
    let inserted = if bit == high then 0x80 else 0x00 :: BitVector 8
        sh = (rxShift s `shiftR` 1) .|. inserted
        s' = s{rxShift = sh, rxCnt = 0}
     in if i == maxBound
          then (s'{rxState = RxStop}, (Nothing, False))
          else (s'{rxState = RxData (i + 1)}, (Nothing, False))
  RxStop
    | bit == high -> (s{rxState = RxIdle, rxCnt = 0}, (Just (rxShift s), False))
    | otherwise -> (s{rxState = RxIdle, rxCnt = 0}, (Nothing, True))
  RxIdle -> (s, (Nothing, False))
 where
  bit = maj (rxS7 s) (rxS8 s) (rxS9 s)
```

```haskell
uartTx tick mbyte = unbundle (mealy txStep initTx (bundle (tick, mbyte)))
 where
  initTx = TxS TxIdle 0 0

txStep :: TxS -> (Bool, Maybe (BitVector 8)) -> (TxS, (Bit, Bool))
txStep s (tick, mbyte) = (s', (line, ready))
 where
  ready = txState s == TxIdle
  line = case txState s of
    TxIdle -> high
    TxStart -> low
    TxData _ -> lsb (txShift s)
    TxStop -> high
  s'
    | TxIdle <- txState s = case mbyte of
        Just b -> s{txState = TxStart, txShift = b, txCnt = 0}
        Nothing -> s
    | not tick = s
    | txCnt s /= maxBound = s{txCnt = txCnt s + 1}
    | otherwise = txAdvance s

txAdvance :: TxS -> TxS
txAdvance s = case txState s of
  TxStart -> s{txState = TxData 0, txCnt = 0}
  TxData i
    | i == maxBound -> s{txState = TxStop, txCnt = 0}
    | otherwise -> s{txState = TxData (i + 1), txShift = txShift s `shiftR` 1, txCnt = 0}
  TxStop -> s{txState = TxIdle, txCnt = 0}
  TxIdle -> s
```
